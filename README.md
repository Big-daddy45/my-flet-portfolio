# DXZ Portfolio

A starter portfolio app built with [Flet](https://flet.dev/) and Python.

## What Is Inside

- `main.py` starts the Flet app and handles navigation.
- `pages/` contains the Home, Projects, Certificates, and Vlogs pages.
- `components/` contains reusable UI pieces such as cards and the navigation bar.
- `assets/` is for images, certificates, and vlog media.

## Run Locally

Install the dependency:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
flet run main.py
```

## What To Customize Next

- Replace `DXZ Portfolio` with your preferred public name or brand.
- Update the About text on the Home page.
- Replace placeholder projects in `pages/projects.py`.
- Add real certificate files to `assets/certificates/`.
- Add screenshots or profile images to `assets/images/`.
- Add vlog thumbnails or clips to `assets/vlogs/`.
